# CerealKillers

## Server-tests Compilation:


0. Clone the server branch with ```git clone -b server https://gitlab.ethz.ch/machart/cerealkillers.git```
1. Create a `build` folder
2. `cd` into `build`
3. cmake .. 
4. make server-tests

### Run test
Run with ```./unit-tests/test_game_controller_tests```
